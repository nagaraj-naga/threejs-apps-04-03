🔀 Load Balancer

Distributing traffic across targets

How It Works

A load balancer distributes incoming requests across multiple targets to ensure no single server is overwhelmed.

Round Robin: Each target gets requests in turn. Least Connections: Targets with fewer active connections get priority. Weighted: Higher-capacity targets get more traffic.
